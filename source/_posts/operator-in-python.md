运算符::的作用

<!-- more -->

 :: 可以看作对切片的一种扩展，::前面的数字代表起始位置，后面的数字代表步长，步长为正则向后遍历，步长为负则向前遍历

```py
>>> l = [1,2,3,4,5,6,7]
>>> l[:1]
[1]
>>> l[::1]
[1, 2, 3, 4, 5, 6, 7]
>>> l[1::2]
[2, 4, 6]
>>> l[1::3]
[2, 5]
>>> l[2::3]
[3, 6]
>>> l[2::-1]
[3, 2, 1]
>>> l[2::-2]
[3, 1]
>>> l[::-2]
[7, 5, 3, 1]
```
